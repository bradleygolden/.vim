Installation:

    git clone https://github.com/bradleygolden/.vim.git
    git clone https://github.com/VundleVim/Vundle.vim.git

Create symlinks:

    ln -s ~/.vim/vimrc ~/.vimrc

Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
